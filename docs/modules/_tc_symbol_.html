Module "tc-symbol"

Module, which contains the definitions of all the TcSymbols and their derived types, through which data manipulation with the Target PLC is made. The TcSymbols act as the interface between the TcContext and the TcBindings, where the TcBindings manage all the data parsing, checking and Symbol memory location, while the TcSymbol itself, as a wrapper for those bindings and collection of children Symbols.
